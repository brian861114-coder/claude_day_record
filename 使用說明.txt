========================================
  每日成長與開發實踐紀錄 - 使用說明
========================================

【簡介】
這是一個用 Flutter 開發的跨平台 App，幫助你每天記錄學習進度、
專案開發、知識輸入、生活健康與每日反思，並將資料自動儲存到 Google Sheets。


【啟動方式】

  1. 開啟終端機，進入專案資料夾：
     cd day_record

  2. 啟動 Web 版：
     flutter run -d web-server --web-port=9090

  3. 在瀏覽器打開：
     http://localhost:9090


【使用流程】

  第1頁 - 首頁
    顯示 App 名稱，點擊「START」開始（首次需要 Google 登入）。

  第2頁 - 今日核心目標
    填寫今天最想達成的一個目標或學習重點。

  第3頁 - 選擇專案
    系統會從 Google Sheets 讀取所有已有的專案名稱。
    - 點擊舊專案 → 自動帶入該專案最近一次的紀錄內容（進入第5頁）
    - 點擊「新專案」→ 建立新的專案紀錄（進入第4頁）

  第4頁 - 新專案
    填寫：項目名稱、進度摘要、技術痛點與解決、
    遇見問題、解決方案、待辦事項。

  第5頁 - 舊專案編輯
    與第4頁相同欄位，但會自動填入上次的紀錄內容。
    修改後會存為新的一筆紀錄（不會覆蓋舊資料）。

  第6頁 - 知識輸入
    填寫：學習主題、關鍵筆記、實踐應用、疑問/待查證。

  第7頁 - 生活與健康
    填寫：飲食紀錄、環境維護。

  第8頁 - 今日反思
    填寫：做得好的地方、明天可以改進的地方。
    點擊「完成」時，空白欄位會自動填入「空白」二字，
    然後整筆資料會寫入 Google Sheets。

  第9頁 - 完成頁
    顯示慶祝訊息，可點擊連結直接開啟 Google Sheets 查看紀錄。


【Google Sheets 欄位說明】

  A. 日期
  B. 今日核心目標
  C. 項目名稱
  D. 進度摘要
  E. 技術痛點與解決
  F. 遇見問題
  G. 解決方案
  H. 待辦事項
  I. 學習主題
  J. 關鍵筆記
  K. 實踐應用
  L. 疑問/待查證
  M. 飲食紀錄
  N. 環境維護
  O. 做得好的地方
  P. 明天可以改進的地方

  表頭會在第一次寫入時自動建立。


【Google Cloud Console 設定（必要）】

  1. 前往 https://console.cloud.google.com
  2. 啟用「Google Sheets API」
  3. OAuth 同意畫面 → 新增範圍：
     https://www.googleapis.com/auth/spreadsheets
  4. 憑證 → Web 用戶端 ID → 已授權的 JavaScript 來源加入：
     http://localhost:9090（或你使用的 port）
